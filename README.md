# 🎮 Jogo da Vida (Simulação em Java)

Este projeto é uma implementação simplificada do clássico "Jogo da Vida", desenvolvido com o objetivo de praticar conceitos de Programação Orientada a Objetos (POO) em Java.

## 📌 Sobre o Projeto

O "Jogo da Vida" é uma simulação baseada em regras simples que determinam a evolução de células em um grid ao longo do tempo. A cada geração, o estado das células muda de acordo com seus vizinhos.

Este projeto foi desenvolvido como exercício prático para reforçar conceitos fundamentais de desenvolvimento backend.

## 🚀 Tecnologias Utilizadas

- Java
- Programação Orientada a Objetos (POO)
- Estruturas de dados básicas (arrays/matrizes)

## 🧠 Conceitos Aplicados

- Encapsulamento
- Abstração
- Manipulação de matrizes
- Lógica de simulação
- Separação de responsabilidades (boas práticas)

## ⚙️ Como Funciona

A aplicação segue as regras clássicas do Jogo da Vida:

1. Uma célula viva com menos de 2 vizinhos morre (solidão)
2. Uma célula viva com 2 ou 3 vizinhos sobrevive
3. Uma célula viva com mais de 3 vizinhos morre (superpopulação)
4. Uma célula morta com exatamente 3 vizinhos nasce

A simulação evolui em "gerações", atualizando o estado do grid a cada iteração.
